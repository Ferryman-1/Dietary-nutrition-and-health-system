# healthy
本系统基于SSM框架开发，辅以HTML，JavaScript进行前端的开发和后端的管理。以对标用户角度完善系统的功能需求，利用 Echarts 可视化图表连接数据库显示数据内容，直观体现各类营养物质的摄入比例，设计用户反馈，利用数据库的增删改查对后台各类模块进行信息的管理与系统的维护。
